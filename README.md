# 目前的问题
## 第一个问题
- 正在想解决方法

必须按图的顺序 从上往下输入节点 的路径

`->` 的左侧一定要在之前出现过，不然会失效

## 第二个问题
打印的箭头方向

- 第一种的显示的结果需要考虑到同级的框，上下层的框和整体的布局。
- 第二种改进：可以把打印输入到一个二维字符串列表里面，先将这个二维字符串列表旋转90度，然后考虑整体布局的情况下将各行居中显示，改变箭头的方向。
